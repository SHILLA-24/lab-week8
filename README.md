## Description
This project demonstrates how to perform CRUD (Create, Read, Update, Delete) operations
in MongoDB using Node.js and Mongoose. The application connects to a local MongoDB
database, defines a schema and model, and executes various database operations
programmatically.

## Features
- Connects to MongoDB using Mongoose
- Creates a schema and model
- Inserts single and multiple documents
- Retrieves documents using find with and without filters
- Counts total documents in a collection
- Updates documents using updateMany
- Deletes documents using deleteMany

## Technologies Used
- Node.js
- MongoDB
- Mongoose
- Visual Studio Code
